# AI Spam Detector
 
A machine learning web app that detects whether a text message is spam or not.
 
**Live demo:** https://ai-spam-detector-z39y.onrender.com
 
## What it does
 
Paste any text message, click Check, and the model tells you if it's spam or ham. Trained on 5,572 real SMS messages with 97.94% accuracy on test data.
 
## Tech used
 
- Python
- scikit-learn — Naive Bayes classifier
- Flask — web server
- pandas — data handling
- HTML/JavaScript — web interface
- Deployed on Render
